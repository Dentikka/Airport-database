# Airport-database

Здесь представлен учебный проект по курсы "Базы данных" в МФТИ. Реализована база "Аэропорт". Работа с ней разделена на несколько лабораторных:

- Проектирование
- Заполнение
- Некоторые запросы
- Представления
- Транзакции
- Контроль доступа
- Метаданные
- Триггеры
- Оптимизация
